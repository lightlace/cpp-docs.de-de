# Eine Stop-Anweisung wurde gefunden.
Eine `Stop`\-Anweisung, die die Ausführung unterbricht, wurde gefunden.  
  
### So beheben Sie diesen Fehler  
  
1.  Entfernen Sie die `Stop`\-Anweisung.
