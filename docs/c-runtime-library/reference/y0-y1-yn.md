# <a name="y0-y1-yn"></a>y0, y1, yn

Die von Microsoft implementierten POSIX-Funktionsnamen `y0`, `y1`und `yn` sind als veraltete Aliase für die Funktionen [_y0, _y1 und _yn](bessel-functions-j0-j1-jn-y0-y1-yn.md) gekennzeichnet. Standardmäßig generieren Sie eine [Compilerwarnung (Stufe 3) C4996](../../error-messages/compiler-warnings/compiler-warning-level-3-c4996.md). Die Namen sind veraltet, da Sie nicht den Standard mäßigen C-Regeln für Implementierungs spezifische Namen folgen. Die Funktionen werden jedoch weiterhin unterstützt.

Es wird empfohlen, stattdessen [_y0, _y1 und _yn zu](bessel-functions-j0-j1-jn-y0-y1-yn.md) verwenden. Oder Sie können diese Funktionsnamen weiterhin verwenden und die Warnung deaktivieren. Weitere Informationen finden Sie unter [Deaktivieren der Namen der Warnungs](../../error-messages/compiler-warnings/compiler-warning-level-3-c4996.md#turn-off-the-warning) -und [POSIX-Funktionen](../../error-messages/compiler-warnings/compiler-warning-level-3-c4996.md#posix-function-names).
