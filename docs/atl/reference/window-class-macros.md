# Window Class Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Diese Makros definieren Fensterklassenhilfsprogramme.  
  
|||  
|-|-|  
|[DECLARE\_WND\_CLASS](../Topic/DECLARE_WND_CLASS.md)|Ermöglicht es, um den Namen einer neuen Fensterklasse anzugeben.|  
|[DECLARE\_WND\_SUPERCLASS](../Topic/DECLARE_WND_SUPERCLASS.md)|Ermöglicht es, um den Namen einer vorhandenen Fensterklasse anzugeben, auf der eine neue Fensterklasse basiert.|  
|[DECLARE\_WND\_CLASS\_EX](../Topic/DECLARE_WND_CLASS_EX.md)|Ermöglicht es Ihnen, die Parameter einer Klasse anzugeben.|
