# Service Map Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Diese Makros definieren und \-Einträge Dienstzuordnungen.  
  
|||  
|-|-|  
|[BEGIN\_SERVICE\_MAP](../Topic/BEGIN_SERVICE_MAP.md)|Markiert den Beginn einer ATL\-Dienstzuordnung.|  
|[END\_SERVICE\_MAP](../Topic/END_SERVICE_MAP.md)|Markiert das Ende einer ATL\-Dienstzuordnung.|  
|[SERVICE\_ENTRY](../Topic/SERVICE_ENTRY.md)|Gibt an, dass das Objekt eine bestimmte ID Dienst unterstützt|  
|[SERVICE\_ENTRY\_CHAIN](../Topic/SERVICE_ENTRY_CHAIN.md)|Weist [IServiceProviderImpl::QueryService](../Topic/IServiceProviderImpl::QueryService.md) an, die im angegebenen Objekt zu verketten.|
