# <a name="error-information-classes"></a>Fehler-Informationsklassen
Die folgende Klasse zeigt an, wie Fehlerinformationen behandelt wird:  
  
-   [ISupportErrorInfoImpl](../atl/reference/isupporterrorinfoimpl-class.md) bestimmt, ob das Objekt unterstützt die [IErrorInfo](http://msdn.microsoft.com/en-us/4dda6909-2d9a-4727-ae0c-b5f90dcfa447) Schnittstelle. **IErrorInfo** Fehlerinformationen zurück an den Client weitergegeben werden können.
