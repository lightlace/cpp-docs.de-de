# <a name="runtimeclassinternaladdref-method"></a>RuntimeClass::InternalAddRef-Methode
Inkrementiert den Verweiszähler dieser Planergruppe mit dem aktuellen RuntimeClass-Objekt.  
  
## <a name="syntax"></a>Syntax  
  
```  
ULONG InternalAddRef();  
```  
  
## <a name="return-value"></a>Rückgabewert  
 Der resultierende Verweiszähler.  
  
## <a name="requirements"></a>Anforderungen  
 **Header:** implements.h  
  
 **Namespace:** Microsoft::WRL
