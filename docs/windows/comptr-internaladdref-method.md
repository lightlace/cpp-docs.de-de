# <a name="comptrinternaladdref-method"></a>ComPtr::InternalAddRef-Methode

## <a name="syntax"></a>Syntax  
  
```  
void InternalAddRef() const;  
```  
  
## <a name="remarks"></a>Hinweise  
 Diese Methode ist geschützt.

 **Namespace:** Microsoft::WRL
