# <a name="ptr"></a>ptr
Kennzeichnet einen Zeiger als vollständige Zeiger.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
[ptr]  
  
```  
  
## <a name="remarks"></a>Hinweise  
 Die **Ptr** C++-Attribut hat die gleiche Funktionalität wie die [Ptr](http://msdn.microsoft.com/library/windows/desktop/aa367149) MIDL-Attribut.  
  
## <a name="example"></a>Beispiel  
 Siehe das Beispiel für ["DefaultValue"](../windows/defaultvalue.md) für ein Beispiel für die Verwendung von **Ptr**.
