# <a name="int2c"></a>__int2c
**Microsoft-spezifisch**  
  
 Generiert die `int 2c` -Anweisung, die Trigger die `2c` Interrupt.  
  
## <a name="syntax"></a>Syntax  
  
```  
void __int2c(void);  
```
