# <a name="label"></a>LABEL

Erstellt eine neue Bezeichnung durch Zuweisen des aktuellen Location-Counter-Werts und des angegebenen *qualifiedtype* zu *Name*.

## <a name="syntax"></a>Syntax

> *namens* **Bezeichnung** *qualifiedtype*
>
> *namens* **Bezeichnung** ⟦**near** | **Far** | **proc**⟧ **ptr** ⟦*qualifiedtype*⟧
