# <a name="fardata"></a>.FARDATA
Bei Verwendung mit [. Modell](../../assembler/masm/dot-model.md), startet ein weit Datensegment für initialisierte Daten (Segmentnamen FAR_DATA oder *Namen*).  
  
## <a name="syntax"></a>Syntax  
  
```  
  
.FARDATA [[name]]  
```
