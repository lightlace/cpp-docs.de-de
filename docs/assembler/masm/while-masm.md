# <a name="while-masm"></a>WHILE (MASM)

Assembly-Blocks wiederholt *Anweisungen* solange *Ausdruck* "true" bleibt.

## <a name="syntax"></a>Syntax

> WÄHREND *Ausdruck*<br/>
> *Anweisungen*<br/>
> ENDM
