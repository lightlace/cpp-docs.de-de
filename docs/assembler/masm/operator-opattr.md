# operator OPATTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Gibt ein Wort zurück, das den Modus und den Bereich des *Ausdrucks*definiert.  Das niedrige Byte an Byte identisch ist, das von [.TYPE](../../assembler/masm/operator-dot-type.md)zurückgegeben wurde.  Das hohe Byte enthält weitere Informationen.  
  
## Syntax  
  
```  
  
OPATTR expression  
```
