# <a name="686p-32-bit-masm"></a>.686p (32-Bit-MASM)

Aktiviert die Assembly aller Anweisungen (einschließlich privilegierter Berechtigungen) für den Pentium Pro-Prozessor. (nur 32-Bit-MASM.)

## <a name="syntax"></a>Syntax

> **.686P**
