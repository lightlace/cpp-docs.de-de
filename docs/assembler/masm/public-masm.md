# <a name="public-masm"></a>PUBLIC (MASM)
Stellt jede Variable, eine Bezeichnung oder eine absolute Symbol angegeben als *Namen* für alle weiteren Module in der Anwendung verfügbar.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
PUBLIC [[langtype]] name [[, [[langtype]] name]]...  
```
