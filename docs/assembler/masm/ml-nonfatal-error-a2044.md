# <a name="ml-nonfatal-error-a2044"></a>Nicht schwerwiegender ML-Fehler A2044
**Ungültiges Zeichen in der Datei**  
  
 Die Quelldatei enthält ein Zeichen außerhalb eines Kommentars, Zeichenfolge, oder literal, wurde nicht erkannt als einen Operator oder ein anderes Zeichen zulässig.
