# <a name="align-masm"></a>ALIGN (MASM)
Richtet die nächste Variable oder Anweisung auf ein Byte, die ein Vielfaches von `number`.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
ALIGN [[  
number  
]]  
  
```
