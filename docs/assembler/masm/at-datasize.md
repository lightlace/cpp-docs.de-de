# <a name="datasize"></a>@DataSize
0 für **sehr klein**, **kleine**, **Mittel**, und **Flatfile** modelliert, 1 für **COMPACT** und  **GROßE** Modelle und 2 für **sehr groß** Modell (numerische entsprechen).  
  
## <a name="syntax"></a>Syntax  
  
```  
  
@DataSize  
  
```
