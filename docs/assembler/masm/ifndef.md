# <a name="ifndef"></a>IFNDEF
Assembly gewährt, wenn *Namen* wurde nicht definiert.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
IFNDEF name  
```  
  
## <a name="remarks"></a>Hinweise  
 Finden Sie unter [IF](../../assembler/masm/if-masm.md) für die vollständige Syntax.
