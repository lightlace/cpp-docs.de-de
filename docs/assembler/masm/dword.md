# <a name="dword"></a>DWORD
Zum Belegen und optional ein Doppelwort (4 Bytes) des Speichers für die einzelnen initialisiert `initializer`.  
  
## <a name="syntax"></a>Syntax  
  
```  
[[name]] DWORD initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>Hinweise  
 Kann auch verwendet werden, als Typspezifizierer an einer beliebigen Stelle ist ein Typ zulässig.
