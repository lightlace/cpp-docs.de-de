# <a name="dword"></a>DWORD

Zugewiesen und initialisiert optional ein Doppelwort (4 Bytes) des Speichers für die einzelnen `initializer`.

## <a name="syntax"></a>Syntax

> [[Name]] DWORD-Initialisierer [[, Initialisierer]]...

## <a name="remarks"></a>Hinweise

Kann auch als Typspezifizierer verwendet werden ein Typ ist überall zulässig.
