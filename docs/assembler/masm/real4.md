# <a name="real4"></a>REAL4
Zum Belegen und optional eine einzelne Gleitkommazahl mit doppelter Genauigkeit (4-Byte) für die einzelnen initialisiert *Initialisierer*.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
name REAL4 initializer [[, initializer]]...  
```
