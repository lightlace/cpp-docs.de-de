# <a name="ml-nonfatal-error-a2054"></a>Nicht schwerwiegender ML-Fehler A2054

 Die bedingte Error-Direktive [. ERRNZ](../../assembler/masm/dot-errnz.md) wurde verwendet, um diesen Fehler zu generieren.
