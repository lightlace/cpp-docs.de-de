# <a name="ml-warning-a4014"></a>ML-Warnung A4014
Anweisungen und initialisierte Daten in Segmenten BSS nicht unterstützt.  
  
 Es wurde versucht, initialisierte Daten innerhalb eines Abschnitts BSS definieren.  Ein Abschnitt BSS wird als eine Klasse definiert, deren Name BSS ist.  Dazu gehören die vereinfachte Segment `.data?`.
