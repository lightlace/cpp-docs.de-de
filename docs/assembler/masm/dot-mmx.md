# <a name="mmx-32-bit-masm"></a>. MMX (32-Bit-MASM)

Aktiviert die Assembly von MMX-Anweisungen oder Anweisungen für eine einfache Anweisung (SIMD). (nur 32-Bit-MASM.)

## <a name="syntax"></a>Syntax

> .MMX
