# <a name="operator-"></a>Operator :
Überschreibt das Standardsegment von *Ausdruck* mit `segment`. Die `segment` kann ein Segment registrieren, Gruppenname, Segmentnamen oder Segment Ausdruck sein. Die *Ausdruck* muss eine Konstante sein.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
segment  
: expression  
```
