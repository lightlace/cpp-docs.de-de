# <a name="ifnb"></a>IFNB
Assembly gewährt, wenn *Textitem* ist nicht leer.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
IFNB textitem  
```  
  
## <a name="remarks"></a>Hinweise  
 Finden Sie unter [IF](../../assembler/masm/if-masm.md) für die vollständige Syntax.
