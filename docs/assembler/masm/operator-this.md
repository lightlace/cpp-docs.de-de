# <a name="operator-this"></a>Operator THIS
Gibt ein Operanden des angegebenen `type` , deren Werte für Offset und das Segment mit dem aktuellen Speicherort Counter-Wert gleich sind.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
THIS type  
```
