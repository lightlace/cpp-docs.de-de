# <a name="operator-mod"></a>Operator MOD
Gibt den ganzzahligen Wert, der den Rest (modulo) zurück, bei einer Division *expression1* von *expression2*.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
expression1 MOD expression2  
```
