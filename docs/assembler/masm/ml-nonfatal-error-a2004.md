# <a name="ml-nonfatal-error-a2004"></a>Nicht schwerwiegender ML-Fehler A2004
**Symbol Typenkonflikt: Bezeichner**  
  
 Die [EXTERNDEF](../../assembler/masm/externdef.md) oder [Bezeichnung](../../assembler/masm/label-masm.md) Richtlinie wurde verwendet, auf eine Variable, Symbol, Datenstruktur oder Bezeichnung, die im selben Modul jedoch mit einem anderen Typ definiert wurde.
