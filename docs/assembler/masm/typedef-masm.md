# <a name="typedef"></a>TYPEDEF

Definiert einen neuen **Typ oder einen** Namen namens *Name*, der dem *Typ* oder der *protodefinition*entspricht.

## <a name="syntax"></a>Syntax

> *Name* des **typedef** - *Typs* | **Proto** *protodefinition*
