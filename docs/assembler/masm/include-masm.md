# <a name="include-masm"></a>INCLUDE (MASM)
Fügt den Quellcode aus der Quelldatei, die vom *Filename* in der aktuellen Quelldatei während der Assembly.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
INCLUDE filename  
```  
  
## <a name="remarks"></a>Hinweise  
 Die *Filename* in spitzen Klammern eingeschlossen werden muss, enthält einen umgekehrten Schrägstrich, Semikolon, größer-als-Zeichen, kleiner-als-Symbol, einfaches Anführungszeichen oder doppelte Anführungszeichen.
