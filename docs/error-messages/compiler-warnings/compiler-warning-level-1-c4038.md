# <a name="compiler-warning-level-1-c4038"></a>Compilerwarnung (Stufe 1) C4038
"Modifizierer": unzulässiger Klassenmodifizierer  
  
 Dieser Modifizierer kann nicht für Klassen mit **dllimport** - oder [dllexport](../../cpp/dllexport-dllimport.md) -Attributen verwendet werden.
