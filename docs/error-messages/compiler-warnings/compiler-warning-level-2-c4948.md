# <a name="compiler-warning-level-2-c4948"></a>Compilerwarnung (Stufe 2) C4948
der Rückgabetyp von 'Accessor' entspricht nicht den letzten Parametertyp der entsprechenden Setter-Methode  
  
Der Compiler hat ein Konflikt zwischen dem Datentyp ist, wird abgerufen, und legen für eine indizierte Eigenschaft gefunden.  
  
C4948 ist nur über die veraltete Compileroption erreichbar **/CLR: oldSyntax**.
