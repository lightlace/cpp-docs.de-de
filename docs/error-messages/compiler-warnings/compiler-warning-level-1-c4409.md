# <a name="compiler-warning-level-1-c4409"></a>Compilerwarnung (Stufe 1) C4409

Unzulässige Anweisungsgröße

Die Anweisung kein Formulars mit der angegebenen Größe. Die kleinste zulässige Größe wurde verwendet.
