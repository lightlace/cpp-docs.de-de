# <a name="compiler-warning-c4961"></a>Compilerwarnung C4961
In 'PGD-Datei' wurden keine Profildaten zusammengeführt, profilgesteuerte Optimierungen werden deaktiviert.  
  
 Es waren keine Profildaten (PGC-Dateien) verfügbar, damit profilgesteuerte Optimierungen durchgeführt werden können.
