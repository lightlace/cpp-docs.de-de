# <a name="compiler-warning-level-1-c4164"></a>Compilerwarnung (Stufe 1) C4164

"Bezeichner": Systeminterne Funktion nicht deklariert

Die angegebene systeminterne Funktion wurde nicht deklariert. Möglicherweise müssen Sie die entsprechende Headerdatei mithilfe von #include einschließen.
