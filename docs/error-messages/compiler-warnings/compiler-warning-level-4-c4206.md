# <a name="compiler-warning-level-4-c4206"></a>Compilerwarnung (Stufe 4) C4206

**nicht dem Standard entsprechende Erweiterung: Übersetzungseinheit ist leer**

Die Datei wurde nach dem Präprozessorlauf leer.

Diese Erweiterung kann verhindern, dass Ihr Code wird auf andere Compiler übertragbar. Erzeugt einen Fehler ANSI-Kompatibilität ([/Za](../../build/reference/za-ze-disable-language-extensions.md)) und gilt nur für C#-Quellcode.
