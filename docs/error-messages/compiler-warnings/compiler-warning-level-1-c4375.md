# <a name="compiler-warning-level-1-c4375"></a>Compilerwarnung (Stufe 1) C4375
nicht öffentlichen Methode "Methode2" überschreibt "Methode1" nicht.  
  
 Ein Typ, der einen anderen Typ implementiert definiert eine Überschreibungsmethode, aber die Außerkraftsetzung war nicht öffentlich. Aus diesem Grund wird die Methode die Basistyp-Methode nicht überschrieben.
