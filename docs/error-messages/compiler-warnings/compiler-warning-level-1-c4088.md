# <a name="compiler-warning-level-1-c4088"></a>Compilerwarnung (Stufe 1) C4088
'Funktion': Zeiger übereinstimmende Datentypen in der tatsächlichen Parameter "Number", formale Parameter 'Nummer'  
  
 Die entsprechenden formale und tatsächliche Parameter besitzen eine andere Dereferenzierungsebene. Ohne Änderung wird der tatsächliche Parameter übergeben. Die aufgerufene Funktion interpretiert seinen Wert als Zeiger.
