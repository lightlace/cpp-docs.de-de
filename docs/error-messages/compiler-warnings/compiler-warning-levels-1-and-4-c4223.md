# <a name="compiler-warning-levels-1-and-4-c4223"></a>Compilerwarnung (Stufen 1 und 4) C4223

nicht dem Standard entsprechende Erweiterung: kein l-Wert-Array in Zeiger konvertiert.

Sie können nicht in standard-C ein l-Array in einen Zeiger konvertieren. Mit den Standard-Microsoft-Erweiterungen ([/Ze](../../build/reference/za-ze-disable-language-extensions.md)), können Sie.
