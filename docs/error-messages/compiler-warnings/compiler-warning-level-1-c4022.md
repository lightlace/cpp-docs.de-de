# <a name="compiler-warning-level-1-c4022"></a>Compilerwarnung (Stufe 1) C4022
'Funktion': Zeiger nicht übereinstimmende für tatsächliche Parameter 'Nummer'  
  
 Der Zeigertyp des übergebenen Parameters unterscheidet sich von der Zeigertyp des entsprechenden formalen Parameters. Ohne Änderung wird der tatsächliche Parameter übergeben.
