# Compilerwarnung (Stufe 1) C4022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Funktion': Zeigertyp für übergebenen Parameter 'Nummer' stimmt nicht mit dem deklarierten Zeigertyp überein  
  
 Der Zeigertyp des übergebenen Parameters unterscheidet sich von dem des entsprechenden formalen Parameters.  Der tatsächliche Parameter wird unverändert übergeben.
