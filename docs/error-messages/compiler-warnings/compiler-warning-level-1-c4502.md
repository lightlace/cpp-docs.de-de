# <a name="compiler-warning-level-1-c4502"></a>Compilerwarnung (Stufe 1) C4502
"Verknüpfungsspezifikation" erfordert die Verwendung des Schlüsselworts "Extern" und muss vor allen anderen Spezifizierern stehen  
  
 Eine Bindung wurde angegeben, ohne die `extern` Schlüsselwort. Verknüpfung ist nicht in Typen nicht Extern relevant.  
  
 Der Compiler davon ausgegangen, dass die `extern` Schlüsselwort.
