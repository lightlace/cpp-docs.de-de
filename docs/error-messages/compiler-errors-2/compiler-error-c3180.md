# <a name="compiler-error-c3180"></a>Compilerfehler Fehler C3180
"Typname": Name mehr als das Metadatenlimit von 'Limit'-Zeichen enthält  
  
 Der Compiler gekürzt, den Namen für einen verwalteten Typ in den Metadaten. Das Abschneiden wird machen Sie den Typ kann nicht verwendet werden, mit der `#using` -Direktive (oder die entsprechenden Werte in einer anderen Sprache).  
  
 Der Typname Grenzwert schließt alle Namespacequalifizierungen.
