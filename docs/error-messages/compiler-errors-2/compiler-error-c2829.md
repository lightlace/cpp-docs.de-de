# <a name="compiler-error-c2829"></a>Compilerfehler C2829

'Operator Operator' kann nicht keine Variablenparameterliste haben.

Nur zwei Operatoren können Variablen Parameterlisten ausführen:

- Funktionsaufruf `()`

- `new`
