# <a name="compiler-error-c3096"></a>Compilerfehler C3096

"Attribut": Das Attribut ist nur für Datenmember von Attributklassen zulässig.

Ein Attribut wurde falsch angewendet.

Weitere Informationen finden Sie unter [User-Defined Attributes](../../extensions/user-defined-attributes-cpp-component-extensions.md).
