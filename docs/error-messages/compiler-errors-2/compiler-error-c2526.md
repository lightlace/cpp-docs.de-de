# <a name="compiler-error-c2526"></a>Compilerfehler C2526

'Bezeichner1': Funktion mit C-Bindung kann nicht C++-Klasse "Bezeichner2" zurückgeben

Eine Funktion mit C-Bindung definiert, kann keinen benutzerdefinierten Typ zurückgeben.
