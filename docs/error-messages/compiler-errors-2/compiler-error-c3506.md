# Compilerfehler C3506
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Kein typelib für LIBID 'Id' registriert  
  
 Ein typelib wurde nicht ordnungsgemäß registriert.  Verwenden Sie regsvr32.exe, um eine Typbibliothek zu registrieren.
