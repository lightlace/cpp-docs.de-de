# Compilerfehler C2941
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Klasse': Typ\-Klassen\-Id wird als lokales 'element' neu definiert  
  
 Eine generische oder Vorlagenklasse kann nicht als Element verwendet werden.
