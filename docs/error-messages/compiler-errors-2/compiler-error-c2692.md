# <a name="compiler-error-c2692"></a>Compilerfehler Fehler C2692
"Funktionsname": vollständige Prototypen Funktionen im C-Compiler mit der "/ Clr" Option  
  
 Beim Kompilieren für .NET Code verwalteten erfordert der C-Compiler Funktionsdeklarationen ANSI. Darüber hinaus, wenn eine Funktion keine Parameter akzeptiert, es muss explizit deklarieren `void` als Parametertyp.
