# <a name="compiler-error-c3240"></a>Compilerfehler C3240
'Funktion': muss eine Funktion abstrakten Member nicht Ã¼berladen von "Type" sein  
  
 Ein Basistyp enthielt eine Funktion, die definiert wurde. Funktion muss virtuell sein.  
  
## <a name="example"></a>Beispiel  
 Im folgende Beispiel wird C3240 generiert.  
  
```  
// C3240.cpp  
// compile with: /c  
__interface I {  
   void f();  
};  
  
struct A1 : I {   
   void f() {}  
};  
  
struct A2 : I {   
   void f() = 0;  
};  
  
template <class T>   
struct A3 : T {  
   void T::f() {}  
};  
  
template <class T>   
struct A4 : T {  
   void T::f() {}  
};  
  
A3<A1> x;   // C3240  
A3<I> x2;  
A4<A2> x3;  
```
