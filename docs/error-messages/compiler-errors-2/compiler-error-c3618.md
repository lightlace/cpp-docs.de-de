# <a name="compiler-error-c3618"></a>Compilerfehler C3618
'Funktion': eine DllImport markierte Methode kann nicht definiert werden  
  
 Eine Methode mit markiert <xref:System.Runtime.InteropServices.DllImportAttribute> ist definiert in der angegebenen. DLL.  
  
## <a name="example"></a>Beispiel  
 Im folgende Beispiel wird C3618 generiert.  
  
```  
// C3618.cpp  
// compile with: /clr /c  
using namespace System;  
using namespace System::Runtime::InteropServices;  
  
[ DllImport("user32.dll", EntryPoint="MessageBox", CharSet=CharSet::Ansi) ]  // CHANGED   
void Func();   
  
void Func() {}   // C3618, remove this function definition to resolve  
```
