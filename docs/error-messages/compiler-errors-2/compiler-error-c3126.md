# <a name="compiler-error-c3126"></a>Compiler-Fehler C3126 generiert
eine Union "Union" innerhalb des verwalteten Typs 'Type' kann nicht definiert werden.  
  
 Eine Union kann nicht innerhalb eines verwalteten Typs definiert werden.  
  
 Im folgende Beispiel wird C3126 generiert:  
  
```  
// C3126_2.cpp  
// compile with: /clr /c  
ref class Test  
{  
   union x  
   {   // C3126  
      int a;  
      int b;  
   };  
};  
```
