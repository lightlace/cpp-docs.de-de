# Compilerfehler C2408
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Unzulässiger Typ in Verbindung mit dem Operator PTR in "context"  
  
 Der erste Parameter des `PTR`\-Operators ist keine gültige Typspezifikation.
