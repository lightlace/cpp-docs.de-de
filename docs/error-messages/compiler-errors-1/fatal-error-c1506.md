# <a name="fatal-error-c1506"></a>Schwerwiegender Fehler C1506
Weiterverarbeitung nach der Auflösung von Blöcken nicht möglich  
  
 Der Block war zu groß für kompilieren.  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Dieser Fehler kann eine der folgenden Ursachen haben:  
  
1.  Nicht übereinstimmende Klammern  
  
2.  Außergewöhnlich große Funktion oder Klasse
