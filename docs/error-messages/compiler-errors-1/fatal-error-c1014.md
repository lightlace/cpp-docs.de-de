# <a name="fatal-error-c1014"></a>Schwerwiegender Fehler C1014

Zu viele Includedateien: Tiefe = Ebene

Die `#include` -Direktiven sind zu tief geschachtelt. Geschachtelte Direktiven können geöffnete Dateien enthalten. Die Quelldatei, die die Direktive enthält, zählt als eine Datei.
