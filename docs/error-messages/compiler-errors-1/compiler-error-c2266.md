# <a name="compiler-error-c2266"></a>Compilerfehler C2266

'identifier': Der Verweis auf ein nicht durch Konstanten begrenztes Array ist unzulässig

Ein Verweis ist für ein Array ohne Begrenzung durch Konstanten deklariert. Das Array muss durch Konstanten begrenzt sein.
