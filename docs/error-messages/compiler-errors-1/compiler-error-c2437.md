# <a name="compiler-error-c2437"></a>Compilerfehler C2437

"Bezeichner": bereits initialisiert

Ein-Objekt kann nur einmal initialisiert werden.

Im folgenden Beispiel wird C2437 generiert:

```cpp
// C2437.cpp
// compile with: /c
class A {
public:
   A(int i) {}
};

class B : A {
   B() : A(1), A(2) {}   // C2437
   B() : A(1) {}   // OK
};
```
