# <a name="compiler-error-c2324"></a>Compilerfehler C2324
'Bezeichner': Unerwartetes rechts von "Name"  
  
 Unter Verwendung eines falschen Bezeichners ein Destruktor aufgerufen.  
  
 Im folgende Beispiel wird C2324 generiert:  
  
```  
// C2324.cpp  
class A {};  
typedef A* pA_t;  
int i;  
  
int main() {  
   pA_t * ppa = new pA_t;  
   ppa->~i;   // C2324  
   ppa->~pA_t();   // OK  
}  
```
