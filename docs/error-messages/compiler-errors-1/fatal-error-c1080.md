# <a name="fatal-error-c1080"></a>Schwerwiegender Fehler C1080

Compilerlimit: Befehlszeilenoption überschreitet die Grenze der Anzahl von Zeichen

Ein Argument an den Compiler übergeben länger als 256 Zeichen.
