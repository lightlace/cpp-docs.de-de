 Ein `__based` Ausdruck muss auf einen Zeiger basieren.  
  
 Im folgende Beispiel wird C2493 generiert:  
  
```  
// C2493.cpp  
// compile with: /c  
char mybase;  
int __based(mybase) ptr;   // C2493  
  
// OK  
char * mybase;  
int __based(mybase) * ptr;  
```
