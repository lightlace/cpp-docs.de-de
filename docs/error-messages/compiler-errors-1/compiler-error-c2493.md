# <a name="compiler-error-c2493"></a>Compilerfehler C2493

Ein `__based` Ausdruck muss auf einen Zeiger basieren.

Im folgende Beispiel wird die C2493 generiert:

```
// C2493.cpp
// compile with: /c
char mybase;

// OK
char * mybase;
int __based(mybase) * ptr;
```
