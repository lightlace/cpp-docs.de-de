# <a name="compiler-error-c2112"></a>Compilerfehler C2112

"-": Subtraktion von Zeigern erfordert Ganzzahl- oder Zeigeroperanden

Es wurde versucht, Zeiger subtrahiert werden soll, die auf verschiedene Typen verweisen.
