# <a name="compiler-error-c2162"></a>Compilerfehler C2162

formaler Makroparameter erwartet

Das Token nach einer Zeichenfolgenoperator (#) ist kein Name formalen Parameter.

## <a name="example"></a>Beispiel

Im folgende Beispiel wird die C2162 generiert:

```
// C2162.cpp
// compile with: /c
#include <stdio.h>

#define print(a) printf_s(b)   // OK
#define print(a) printf_s(#b)    // C2162
```
