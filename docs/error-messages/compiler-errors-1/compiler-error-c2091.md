# <a name="compiler-error-c2091"></a>Compilerfehler C2091

Funktion gibt Funktion zurück.

Eine Funktion kann nicht mit eine Funktion zurückgeben. Geben Sie stattdessen einen Zeiger auf eine Funktion zurück.
