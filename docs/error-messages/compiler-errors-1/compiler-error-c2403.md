# <a name="compiler-error-c2403"></a>Compilerfehler C2403
'bezeichner' : Das Register muss in 'context' Basis/Index sein.  
  
 Das in einem indirekten Arbeitsspeicheroperanden verwendete Register muss in diesem Kontext ein Basis- oder Indexregister sein.
