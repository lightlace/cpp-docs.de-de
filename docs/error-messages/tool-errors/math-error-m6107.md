# <a name="math-error-m6107"></a>Mathematischer Fehler M6107
nicht emuliert  
  
 Es wurde versucht, eine Coprozessoranweisung ausführen, ist ungültig oder wird vom Emulator nicht unterstützt.  
  
 Programm mit Exitcode 135 beendet wird.
