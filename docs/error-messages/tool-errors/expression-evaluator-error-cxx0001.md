# <a name="expression-evaluator-error-cxx0001"></a>Ausdrucksauswertungsfehler CXX0001
Ausnahme ausgeführten Benutzer-Funktion  
  
 Der Code ausgeführt wird, verursacht eine allgemeine schutzverletzung. Dieser Fehler ist mit CAN0001 identisch.
