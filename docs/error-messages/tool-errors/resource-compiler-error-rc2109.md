# <a name="resource-compiler-error-rc2109"></a>Ressourcencompiler: Fehler RC2109
numerische Dialogfeldkonstante erwartet  
  
 Ein **Dialogfeld** Anweisung erfordert ganzzahlige Werte für die *X, y, Breite*, und *Höhe* Felder. Stellen Sie sicher, dass diese Werte nach der **Dialogfeld** -Schlüsselwort und nicht negativ sind.
