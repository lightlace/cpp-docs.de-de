# <a name="resource-compiler-fatal-error-rw1023"></a>Ressourcencompiler: Schwerwiegender Fehler RW1023
E/a-Fehler beim Schreiben der Datei, Laufwerk voll  
  
 Freier Speicherplatz muss über mindestens zwei Mal die Größe der ausführbaren Datei entsprechen, die Sie erstellen.
