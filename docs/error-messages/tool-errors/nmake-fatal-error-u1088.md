# <a name="nmake-fatal-error-u1088"></a>NMAKE: Schwerwiegender Fehler U1088
Ungültiges Trennzeichen "::" auf Rückschlussregel  
  
 Eine Rückschlussregel muss ein einfacher Doppelpunkt folgen (**:**).
