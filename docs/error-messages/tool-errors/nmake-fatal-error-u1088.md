# <a name="nmake-fatal-error-u1088"></a>NMAKE: Schwerwiegender Fehler U1088

Ungültiges Trennzeichen "::" in Rückschlussregel

Eine Rückschlussregel muss durch einen einzigen Doppelpunkt folgen (**:**).
