# <a name="expression-evaluator-error-cxx0018"></a>Ausdrucksauswertungsfehler CXX0018
Ungültige RegisterName  
  
 Das angegebene Register ist nicht vorhanden oder kann nicht angezeigt werden.  
  
 Das Fenster "überwachen" kann die folgenden Register anzeigen:  
  
 AX SP DS IP  
  
 BX BP VORANGESTELLTEN FL  
  
 CX SI SS GS  
  
 DX DI CS SS  
  
 EAX ESP DS EIP  
  
 EBX EBP VORANGESTELLTEN EFL  
  
 ECX ESI FS  
  
 DIE EDI-CS EDX  
  
 Dieser Fehler ist mit CAN0018 identisch.
