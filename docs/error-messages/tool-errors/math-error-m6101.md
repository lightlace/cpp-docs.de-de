# <a name="math-error-m6101"></a>Mathematischer Fehler M6101

Ungültig

Ungültiger Vorgang.

Dieser Fehler kann verursacht werden, wenn ein Operand NaN (keine Zahl) oder unendlich ist.

Das Programm mit Exitcode 129 wird beendet.
