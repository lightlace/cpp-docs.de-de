# <a name="project-build-error-prj0046"></a>Projektbuildfehler PRJ0046

Über die Befehlszeile konnte nicht erzeugt werden, weil die Angabe leer war.

Für ein Makefile-Konfiguration wurde eine leere Befehlszeile angegeben; die Befehlszeile ist erforderlich.
