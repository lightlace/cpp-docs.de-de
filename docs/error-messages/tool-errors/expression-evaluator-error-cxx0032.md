# Ausdrucksauswertungsfehler CXX0032
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Division durch Null  
  
 Der Ausdruck enthält eine Division durch 0 \(null\), die ungültig ist.  Der Divisor besteht entweder aus der Zahl 0 \(null\), oder er besteht aus einem Ausdruck, der zu 0 \(null\) ausgewertet wird.  
  
 Dieser Fehler ist mit CAN0032 identisch.
