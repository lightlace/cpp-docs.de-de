# <a name="resource-compiler-error-rc2112"></a>Ressourcencompiler: Fehler RC2112
BEGIN in Dialogfeld erwartet.  
  
 Die **beginnen** -Schlüsselwort muss unmittelbar folgen der **Dialogfeld** Schlüsselwort.
