# Linkertoolwarnung LNK4065
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Funktion' kann nicht geordnet werden; ignoriert  
  
 Die angegebene Funktion wurde nicht als Paketfunktion kompiliert.  Kompilieren Sie mithilfe von [\/Gy](../../build/reference/gy-enable-function-level-linking.md) neu.
