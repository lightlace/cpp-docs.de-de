# <a name="expression-evaluator-error-cxx0061"></a>Ausdrucksauswertungsfehler CXX0061

Funktion Schachtelungstiefe überschritten

Der Ausdruck enthält eine Funktion, die Schachtelungstiefe ist größer als der Grenzwert. Ändern Sie den Ausdruck aus, um die Schachtelungstiefe zu reduzieren.

Dieser Fehler ist mit CAN0061 identisch.
