# <a name="project-build-error-prj0040"></a>Projektbuildfehler PRJ0040

Interner Fehler bei der Erstellung. Kann nicht fortgesetzt werden. Projekt erneut laden, und versuchen Sie es erneut.

Das Projektsystem konnte das Projekt zu verarbeiten.

Laden Sie das Projekt erneut.
