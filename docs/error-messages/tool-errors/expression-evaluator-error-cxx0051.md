# <a name="expression-evaluator-error-cxx0051"></a>Ausdrucksauswertungsfehler CXX0051

mehr als ein überladenes Symbol angegeben wird, im Haltepunkt

Die ausdrucksauswertung konnte die von mehr als ein überladenes Symbol für die Verwendung als Haltepunkt nicht ermitteln.

Dieser Fehler ist mit CAN0051 identisch.
