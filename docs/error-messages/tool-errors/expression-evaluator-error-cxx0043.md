# Ausdrucksauswertungsfehler CXX0043
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"This"\-Zeiger außerhalb der Memberfunktion verwendet  
  
 Der **this**\-Zeiger kann nur für nicht statische Memberfunktionen verwendet werden.  
  
 Dieser Fehler ist mit CAN0043 identisch.
