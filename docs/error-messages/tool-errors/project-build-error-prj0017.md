# <a name="project-build-error-prj0017"></a>Projektbuildfehler PRJ0017
Das aktuelle Arbeitsverzeichnis ist ungültig.  
  
 Der Pfad zum aktuellen Arbeitsverzeichnis ist länger als durch _MAXPATH vorgegeben. Erstellen Sie das Projekt nicht auf tiefer Ebene, um diesen Fehler zu beheben.
