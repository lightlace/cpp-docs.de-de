# <a name="project-build-error-prj0017"></a>Projektbuildfehler PRJ0017

Das aktuelle Arbeitsverzeichnis ist ungültig.

Der Pfad zu das aktuelle Arbeitsverzeichnis ist länger als durch _MAXPATH vorgegeben. Erstellen Sie Ihr Projekt nicht auf solche eine detaillierte Ebene, um diesen Fehler zu beheben.
