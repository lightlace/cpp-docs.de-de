# <a name="expression-evaluator-error-cxx0062"></a>Ausdrucksauswertungsfehler CXX0062
Konstruktoraufrufe nicht unterstützt  
  
 Ein Ausdruck einen Aufruf eines Konstruktors. Ausdrücke nicht explizite Aufrufe an Konstruktoren oder Konvertierungen, die einen Aufruf eines Konstruktors erfordern.  
  
 Dieser Fehler ist mit CAN0062 identisch.
