# <a name="resource-compiler-warning-rc4214"></a>Ressourcencompiler: Warnung RC4214

Codepage ungültig: wird ignoriert.

Die RC-Datei enthielt ein Codepage-Argument, und die angegebene Codepage ist ungültig. Weitere Informationen finden Sie unter [IsValidCodePage](/windows/win32/api/winnls/nf-winnls-isvalidcodepage) .
