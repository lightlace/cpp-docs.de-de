# <a name="null-directive"></a>NULL-Anweisung
Die Präprozessordirektive null ist ein einzahlzeichen (**#**) allein in einer Zeile. Sie hat keine Auswirkungen.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
#  
  
```
