# <a name="namedguids"></a>named_guids
**C++-spezifisch**  
  
 Teilt dem Compiler mit definieren und initialisieren die GUID-Variablen im alten Stil des Formulars **LIBID_MyLib**, **CLSID_MyCoClass**, **IID_MyInterface**, und **DIID _MyDispInterface**.  
  
## <a name="syntax"></a>Syntax  
  
```  
named_guids  
```
