# <a name="internal-classes"></a>Interne Klassen
Die folgenden Klassen werden in MFC intern verwendet. Vollständigkeitshalber werden diese internen Klassen in diesem Abschnitt beschreiben. Sie sollen allerdings nicht direkt im Code verwendet werden.  
  
## <a name="in-this-section"></a>In diesem Abschnitt  
  
|Klasse|Beschreibung|  
|-----------|-----------------|  
|`CCommandManager Class`|Die `CCommandManager` Klasse verwaltet, Befehle und ihre Zuordnung mit Bildern.|  
|`CDialogImpl Class (MFC)`|Erhalten Sie die Implementierungsdetails werden für Box-basierten Klassen, wie z. B. `CDialogEx`.|  
|`CDocumentAdapter Class`|Implementiert die IDocument-Schnittstelle, die zum Suchen und Organisieren Handler erforderlich.|  
|`CFrameImpl Class`|Die `CFrameImpl` Klasse behandelt symbolleistenanpassung für die folgenden Klassen für Rahmenfenster: [CFrameWndEx Klasse](../../mfc/reference/cframewndex-class.md), [CMDIFrameWndEx-Klasse](../../mfc/reference/cmdiframewndex-class.md), und [COleIPFrameWndEx Klasse](../../mfc/reference/coleipframewndex-class.md).|  
|`CFullScreenImpl Class`|Implementiert Vollbild-Funktionalität für SDI und MDI-Anwendungen.|  
|`CMDIClientAreaWnd Class`|Die `CMDIClientAreaWnd` -Klasse ist eine Hilfsklasse, die die Implementierung der MDI-Registerkarten und im Registerkartenformat MDI-Gruppen vereinfacht.|  
|`CMemDC Class`|Eine Hilfsklasse für einen Gerätekontext Arbeitsspeicher. Die Speichergerätekontext unterstützt Zeichnung.|  
|`CMenuHash Class`|Liest den Status der `CMFCToolBar` -Objekte aus einem Archiv oder schreibt den Status in ein Archiv.|  
|`CMenuImages Class`|Stellt die Funktionalität können Sie vordefinierte Images, z. B. Schließ-Schaltflächen anzeigen zu maximieren, Schaltflächen, Optionsfelder und Pfeile.|  
|`CMFCCaptionButtonEx Class`|Enthält den clientfremden Bereich von Schaltflächen mit der Beschriftung an.|  
|`CMFCCaptionMenuButton Class`|Stellt eine Menüschaltfläche, die im Bereich "Caption" eines Aufgabenbereichs oder einer Symbolleiste befindet.|  
|`CMFCColorPropertySheet Class`|Verwendet von [CMFCColorDialog Klasse](../../mfc/reference/cmfccolordialog-class.md) um ein Dialogfeld im Registerformat bereitzustellen.|  
|`CMFCControlBarImpl Class`|Bietet die Implementierungsdetails werden für die Leiste-basierten Klassen zu steuern, wie z. B. `CMFCReBar` und `CMFCToolBar`.|  
|`CMFCControlRenderer Class`|Eine Hilfsklasse, die Image-Rendering behandelt.|  
|`CMFCControlRendererInfo Class`|Eine Hilfsklasse zum Übergeben von Parametern zu `CMFCControlRenderer` Klasse.|  
|`CMFCCustomizeButton Class`|Unterstützt einen Menübefehl oder eine Symbolleisten-Schaltfläche, die ein Anpassungsdialogfelds beginnt.|  
|`CMFCCustomizeMenuButton Class`|Stellt eine einzelne Schaltfläche in einem Menü Anpassung dar.|  
|`CMFCDropDownListBox Class`|Bietet Funktionen zum Menüband-Kombinationsfelder Dropdown-Liste.|  
|`CMFCMousePropertyPage Class`|Eine Eigenschaftenseite, die dem Benutzer ermöglicht, das Verhalten der Maus anpassen.|  
|`CMFCOutlookBarPaneAdapter Class`||  
|`CMFCOutlookBarPaneButton Class`|Implementiert eine Schaltfläche, die das Framework hinzufügt ein `CMFCOutlookBarPane` Objekt.|  
|`CMFCOutlookBarPaneList Class`|Erweitert die `CMFCOutlookBarPane` -Klasse, die Outlook-Stil bereitzustellen.|  
|`CMFCOutlookBarScrollButton Class`|Stellt die Bildlauffunktionen zu den `CMFCOutlookBarTabCtrl` Klasse.|  
|`CMFCOutlookBarToolBar Class`|Implementiert eine Symbolleiste, die den visuellen Stil von Outlook 2003.|  
|`CMFCPropertySheetCategoryInfo Class`|Stellt einen Knoten in einem Strukturansicht-Steuerelement dar.|  
|`CMFCPropertySheetTabCtrl Class`|Erweitert die `CMFCTabCtrl` -Klasse, Eigenschaftenblatt Funktionalität bereitzustellen.|  
|`CMFCProperySheetListBox Class`|Erweitert die `CListBox` -Klasse, Eigenschaftenblatt Funktionalität bereitzustellen.|  
|`CMFCReBarState Class`||
