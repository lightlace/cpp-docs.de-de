# <a name="summary-of-string-literals"></a>Zusammenfassung von Zeichenfolgenliteralen
*string-literal*:  
 **'** *s-char-sequence* opt **'**  
  
 **L'** *s-char-sequence* opt **'**  
  
 *s-char-sequence*:  
 *s-char*  
  
 *s-char-sequence s-char*  
  
 *s-char*:  
 Alle Elemente des Quellzeichensatzes mit Ausnahme von doppelten Anführungszeichen ("), umgekehrtem Schrägstrich (\\) oder Zeilenumbruchzeichen *Escapesequenz*
