# <a name="endcolumnmap"></a>END_COLUMN_MAP
Markiert das Ende der Spalte-Zuordnungseinträge.  
  
## <a name="syntax"></a>Syntax  
  
```cpp
END_COLUMN_MAP()  
  
```  
  
## <a name="remarks"></a>Hinweise  
 Es wird mit einer einzelnen Zugriffsmethode in einem Rowset verwendet. Das `BEGIN_COLUMN_MAP` -Makro wird mit dem `END_COLUMN_MAP` -Makro abgeschlossen.

 Finden Sie unter [BEGIN_COLUMN_MAP](../../data/oledb/begin-column-map.md).
