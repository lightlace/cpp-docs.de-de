# CAccessorRowset::Close
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Gibt alle aktiven Accessoren und das Rowset frei.  
  
## Syntax  
  
```  
  
void Close( );  
  
```  
  
## Hinweise  
 Gibt jeden zugeordneten Arbeitsspeicher frei.

 **Header:** atldbcli.h
