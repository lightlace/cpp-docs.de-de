# <a name="caccessorrowsetclose"></a>CAccessorRowset::Close
Gibt alle aktiven Accessoren und Rowsets frei.  
  
## <a name="syntax"></a>Syntax  
  
```cpp
void Close();  
  
```  
  
## <a name="remarks"></a>Hinweise  
 Gibt alle zugeordneten Arbeitsspeicher frei.

 **Header:** atldbcli.h
